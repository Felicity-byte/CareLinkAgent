## 常见问题

### 1. 端口被占用

服务启动时会自动清理占用端口的进程。如果仍有问题，手动清理：

```powershell
# 查找占用端口的进程
Get-NetTCPConnection -LocalPort 50053

# 终止进程（替换 PID）
Stop-Process -Id <PID> -Force
```

### 2. 环境变量未设置

确保已设置 `GLM_API_KEY` 环境变量：

```powershell
# 临时设置（当前会话）
$env:GLM_API_KEY = "your_api_key"

# 永久设置
[Environment]::SetEnvironmentVariable("GLM_API_KEY", "your_api_key", "User")
```

### 3. 依赖缺失

安装所需依赖：

```powershell
cd "D:\Trea\medical systems\CareLinkAgent\GlmAI"
pip install -r requirements.txt
```
