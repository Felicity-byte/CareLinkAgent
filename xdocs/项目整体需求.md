## 一、项目概述

CareLinkAgent 是一个基于 AI 的智能医疗预诊系统，专注于术后随访场景。系统通过 AI 对话收集患者术后恢复信息，自动生成结构化病历报告，辅助医生进行远程随访管理。

### 核心功能

| 功能模块 | 描述 |
| --- | --- |
| AI 智能问诊 | 基于大语言模型的多轮对话，自动收集患者症状信息 |
| 结构化病历生成 | 自动生成包含现病史、症状、生命体征等字段的病历报告 |
| RAG 知识检索 | 基于向量数据库的医疗知识增强检索 |
| 用户管理 | 患者注册、登录、信息绑定 |
| 预约挂号 | 科室选择、医生预约 |
| 历史记录 | 对话历史、病历查询 |

## 五、核心模块说明

### 5.1 AI对话服务 (GlmAI)

**核心能力**:
- 多轮对话上下文管理
- 流式输出响应
- RAG知识增强检索
- 结构化病历生成

**工作流程**:
```
用户消息 → 会话管理 → RAG检索 → LLM生成 → 流式输出
                ↓
           结束会话 → 病历生成 → 返回报告
```

**gRPC接口**:
| 接口 | 功能 |
| --- | --- |
| CreateSession | 创建会话，返回欢迎语 |
| Chat | 流式对话，支持多轮交互 |
| EndSession | 结束会话，生成病历报告 |
| GetSessionHistory | 获取对话历史 |

### 5.2 后端服务 (Backend)

**核心能力**:
- 用户认证与授权 (JWT)
- 医生/患者信息管理
- 预约挂号管理
- AI服务调用

**API路由**:
| 路由前缀 | 功能 |
| --- | --- |
| /user | 用户注册、登录、信息管理 |
| /doctor | 医生信息、排班管理 |
| /department | 科室列表、信息查询 |

### 5.3 前端应用 (Frontend)

**页面模块**:
| 页面 | 功能 |
| --- | --- |
| Landing | 首页介绍 |
| Login/Register | 用户认证 |
| Chat | AI对话界面 |
| Appointment | 预约挂号 |
| HistoryList | 历史记录 |
| Dashboard | 医生工作台 |
