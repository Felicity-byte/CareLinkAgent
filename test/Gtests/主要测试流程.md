# AI服务测试流程

## 一、环境准备

### 1. 安装Python依赖

```bash
cd D:\Trea\medical systems\CareLinkAgent\GlmAI
pip install -r requirements.txt
```

### 2. 配置环境变量

复制 `.env.example` 为 `.env`，填入API密钥：

```bash
copy .env.example .env
```

编辑 `.env`，填入：

```
ZHIPU_API_KEY=你的智谱API密钥
```

## 二、启动AI服务

### 方法一：新开终端启动

```bash
cd D:\Trea\medical systems\CareLinkAgent\GlmAI
```

看到 `AI服务启动成功!` 表示成功。

## 六、常见问题

### 1. 连接失败

```
[ERROR] 无法连接AI服务
```

**解决**：确保AI服务已启动，端口50053未被占用。

### 2. API密钥错误

```
[ERROR] API密钥无效
```

**解决**：检查 `.env` 文件中的 `ZHIPU_API_KEY` 是否正确。

### 3. 依赖缺失

```
ModuleNotFoundError: No module named 'xxx'
```

**解决**：

```bash
pip install xxx
```
