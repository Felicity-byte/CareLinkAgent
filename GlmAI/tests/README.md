# AI 服务测试文件说明

本目录包含术后随访 AI 服务的所有测试脚本。

## 测试文件列表

| 文件名 | 用途 | 推荐场景 |
|--------|------|----------|
| `test_streaming.py` | **主要测试脚本** - 完整功能测试，包含功能测试、边界测试、错误处理、性能测试 | 日常功能验证 |
| `test_complete.py` | 完整功能测试 - 流式输出演示、结构化病历生成、多轮对话 | 详细功能演示 |
| `test_full_client.py` | 逐步测试所有功能 - 创建会话、对话、获取历史、结束会话 | 分步调试 |
| `test_minimal.py` | 最小化测试 - 仅测试模块导入和基本连接 | 环境验证 |
| `test_client.py` | 基础测试客户端 - 简单的会话和对话测试 | 快速验证 |

## 运行方式

在 `GlmAI` 目录下运行：

```powershell
# 主要测试（推荐）
python tests/test_streaming.py

# 完整功能测试
python tests/test_complete.py

# 逐步测试
python tests/test_full_client.py

# 最小化测试
python tests/test_minimal.py

# 基础测试
python tests/test_client.py
```

## 注意事项

1. 运行测试前请确保 AI 服务已启动（端口 50053）
2. 启动服务命令：`python connect/server.py`
3. 测试需要配置 `GLM_API_KEY` 环境变量
